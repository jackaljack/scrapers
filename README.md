# scrapers
[![Build Status](https://travis-ci.org/jackaljack/scrapers.svg?branch=master)](https://travis-ci.org/jackaljack/scrapers)

Various examples of web scraping techniques in Python 3.5, mostly seen in "Web scraping with Python" (which covers Python 2.7).

Libraries used:

- Beautiful Soup 4
- lxml
- Requests
- RoboBrowser
- Scrapy
- Selenium
